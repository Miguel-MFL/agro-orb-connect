# Script de Consulta de Dados

Este script permite consultar os dados do banco de dados diretamente da pasta do projeto.

## Como usar:

1. **Execute o script:**
   ```bash
   node scripts/consultar-dados.js
   ```

2. O script mostrará:
   - Todos os usuários cadastrados (emails e nomes)
   - Todas as máquinas cadastradas (detalhes completos)

## Observações:

- O script usa as credenciais do Lovable Cloud que já estão configuradas
- Você pode executar quantas vezes quiser
- Os dados são apenas consultados, nada é modificado
